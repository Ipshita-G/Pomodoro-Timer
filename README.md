# 🍅 Pomodoro Timer

A simple Pomodoro Timer built with HTML, CSS, and JavaScript.

This was one of my first JavaScript projects and helped me understand how countdown timers work using `setInterval()` and DOM manipulation.

## Features

- ⏱️ 25-minute countdown
- ▶️ Start timer
- ⏹️ Stop timer
- ⏰ Live minute and second updates

## Technologies Used

- HTML
- CSS
- JavaScript

## What I Learned

- Selecting elements using `querySelector()`
- Handling button clicks with `addEventListener()`
- Using `setInterval()` and `clearInterval()`
- Converting total seconds into minutes and seconds
- Using `Math.floor()` and the modulus (`%`) operator
- Basic debugging and fixing DOM-related issues

## Future Improvements

- ⏸️ Pause/Resume
- 🔄 Reset button
- 🎵 Alarm sound when timer ends
- ⚙️ Custom work/break durations
